TO RUN APPLICATION USE

*JWT SIMPLE*

``
go run .\main.go .\middleware.go
``



**UNIT TEST**

run all test
``
go test
`` 

run all test with verbose
``
go test -v
``

run single test 
``
go test -v -run filename.go
``

run test all package 
``
go test -v ./...
``
